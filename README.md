# Better React Boilerplate

React, Redux, full boilerplate

## Stack

- React
  - react-css-modules
  - Redux
- Webpack
- normalize.css
- ESLint
  - StandardJS base
  - using prettier and pretter-eslint

## Installation

```bash
$ yarn # Install deps
$ cp .env.example .env # Copy Dotenv file
$ yarn start # Launch webpack-dev-server
```
